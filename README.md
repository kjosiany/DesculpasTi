# 🎰 Maracutaia TI: Plantão da Gambiarra

Um utilitário web de alta performance (ou quase) desenvolvido nas profundezas da insônia para fornecer justificativas místicas instantâneas quando o ambiente de produção decidir ir de arrasta pra cima. Ideal para ganhar aqueles 15 minutos preciosos enquanto você reinicia o servidor ou reza para o banco de dados voltar à vida.

---

## 🚀 Funcionalidades Homologadas

* **Tensão Dramática de 3 Segundos:** O algoritmo faz um "spinning" rápido simulando o desespero de uma roleta russa de desculpas antes de entregar o veredito.
* **Interface Anti-Queima de Retina:** Design escuro inspirado nos temas mais amados da madrugada (Drácula/One Dark Pro). Confortável para os olhos cansados de quem está à base de café frio.
* **Banco de Dados de Maracutaias:** Mais de 25 respostas altamente criativas (e altamente não recomendadas para uso real com o cliente {eu nao me responsabilzo KKKK}).

## 🛠️ Tecnologias Utilizadas

Para garantir que o projeto rodasse até em uma torradeira sem dar B.O., a arquitetura foi mantida estritamente minimalista:

* **HTML5** (Estrutura bruta)
* **CSS3** (Visual com apoio da IA)
* **JavaScript** (A lógica por trás do caos)
* **Vercel** (Hospedagem rápida para espalhar a palavra da gambiarra)
